lxc-gentoo

Generates Gentoo guests for use within lxc (Linux Containers) environments.

Dependencies

lxc-utils, and a running kernel with the relevant options enabled (use lxc-checkconfig to test).

Install

lxc-gentoo can be used with the lxc-create script, or independently.
